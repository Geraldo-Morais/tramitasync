# INSS Automação BPC

Extensão Chrome para automatizar o requerimento do BPC no atendimento.inss.gov.br.

## Instalação
1. Abra o Chrome e acesse chrome://extensions
2. Ative o modo desenvolvedor
3. Clique em "Carregar sem compactação" e selecione a pasta `inss-automation-extension`

## Uso
- Abra o site https://atendimento.inss.gov.br
- Clique no ícone da extensão
- Preencha os dados e clique em "Iniciar Automação"

A automação seguirá o fluxo do BPC conforme descrito.

## Observações
- Aguarda carregamentos automaticamente
- Tenta contornar bloqueios comuns de automação
- Permite escolher cidade (Itapetinga ou Vitória da Conquista)
- Adapte o script conforme mudanças no site
